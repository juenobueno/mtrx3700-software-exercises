
This directory contains files for the quick-start MPLAB x / ICD 3 tutorial. 

To use them,
  1. Copy all files to a directory in your user filespace and 
     make them writable. 
  2. Open MPLAB X and create an assembler project in your user filespace.
  3. Add the following file to the project:
      - TUT452.asm     (assembler source file)
  4. Build the project, download to the PICDem 2 board using the ICD 3 debugger
  5. Debug.

For more detailed instructions, see the MPLABX/ICD 3 Tutorial 
instruction document on the MTRX3700 website.

You should also inspect the contents of the file P18F452.inc
